## Установка приложения

> [!IMPORTANT]
>
> Приложение работает только при запущенном клиенте League of Legends, так как использует Live Client Data API.

1. Скачать .exe-файл со [страницы последнего релиза](https://github.com/daryzhk/Lol-Stats-Overlay/releases/tag/v1.0) и запустить его
2. Начать матч в игре
3. В настройках игры, в разделе "видео", поставить в пункте Оконный режим "Без рамок"/"В окне", иначе окно приложения не будет отображаться.


## Описание
Десктопное приложение с веб-интерфейсом для отображения ключевых характеристик чемпиона в игре League of Legends в реальном времени.
Приложение получает данные напрямую из игрового клиента и визуализирует их в удобном интерфейсе без использования браузера. Изначально разработано для чемпиона Смолдер с учётом его игровых характеристик и механик.

## Основные функции
- Отображение характеристик чемпиона:
  - урон, критический шанс и критический урон
  - скорость атаки, лайфстил
  - пробивание брони
  - дальность атаки
- Расчёт итогового урона с учётом критических множителей
- Отображение кулдаунов способностей (Q, E)
- Расчёт CDR (снижение перезарядки)
- Автоматическое обновление данных каждые 2 секунды
- Overlay (отображение поверх других окон)
## Используемые технологии
- Python
- Flask
- JavaScript (fetch API)
- HTML / CSS
- Riot Games API (Live Client Data API)
- PyWebView (desktop-обёртка)
## Работа с API
Приложение использует [Riot Games API (Live Client Data API)](https://developer.riotgames.com/docs/lol#game-client-api_live-client-data-api), предоставляющий доступ к данным игрового клиента в реальном времени.

**Реализовано:**
- Выполнение HTTP-запросов к локальному API клиента игры (127.0.0.1:2999)
- Обработка JSON-ответов
- Извлечение и преобразование игровых параметров
- Работа с данными в реальном времени
## Архитектура
Приложение построено по клиент-серверной архитектуре:

**Backend (Flask):**
- получение данных из API
- обработка и расчёт игровых показателей
- REST endpoint /data

**Frontend:**
- динамическое обновление данных через Fetch API
- отображение статистики в интерфейсе

**Desktop-обёртка:**
- запуск локального сервера
- отображение интерфейса в отдельном окне через PyWebView
- возможность запуска как .exe приложения
## Особенности реализации
- Расчёт кулдаунов способностей с учётом ability haste
- Обработка ошибок при недоступности API (например, если игра не запущена)
- Асинхронное обновление данных без перезагрузки страницы
- Отображение приложения поверх других окон
## Безопасность
- API-ключи не хранятся в открытом виде  
- Используется локальный API клиента  
- Обработка ошибок сети
